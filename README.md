This project is a completed coursework in the subject "Database Systems".
The application "AutoPark" was developed to manage route sheets, drivers and cars by dispatchers.
Relational PostgreSQL and Redis key/value were used.
Backend: Python, Flask.
The interface was built on Bootstrap. Navigation is linear and hierarchical.

How to start:
1) You need Docker + Git
2) git clone ...
   cd CourseWorkAuto
3) docker-compose up --build -d
